# 币安在英国可以使用吗？英国如何使用币安？

2026年10月17日，有报道称数字货币交易所币安已停止接受英国新用户的注册。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/0e1ad80bd0_1706849751-1.jpg) 

我们需要明确什么是英国的新用户定义，许多人对此可能不够清楚。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/e4b20fc74d_1706849777-2.jpg) 

币安仅限制持有英国身份认证的用户注册。

币安不限制位于英国的用户，即IP地址在英国但使用其他地区证件认证的情况。例如，如果您在英国但持有马来西亚证件，可以使用该证件认证币安交易所。

还需注意，身份证件必须与出入金的法定货币保持一致。例如，中国证件只能使用CNY进行出入金，不支持GBP；韩国证件只能使用KRW，不支持英镑。因此，关键要求是证件不能为英国签发。

首先，确保IP地址避开英国、美国、加拿大和新加坡这四个地区。使用这些IP时，系统会提示不服务该区域。建议将IP改为台湾，这是最宽松的地区。

然后，您需要注册一个币安账户。

# **一、创建币安账户**

1. 先将IP改为台湾，然后在Apple Store中搜索“binance”，找到币安交易平台并点击“打开”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/f21235dfd1_1706850147-3.jpg) 

2. 选择“通过邮箱/手机号码注册”，避免使用英国手机号，直接输入您的邮箱。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/72b1ff62f0_1706850885-5.jpg) 

手机号绑定不是必需的，只需用邮箱注册币安账户，再绑定谷歌验证器即可正常使用。

3. 可使用任意邮箱，如Outlook邮箱。填写邀请码[36457687](https://accounts.binance.com/zh-CN/register?ref=36457687)可永久获得20%手续费减免；不填写则无优惠。点击“下一步”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/aca690fd14_1706850831-4.jpg) 

4. 前往邮箱查找验证码，输入后点击“提交”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/157d058ec2_1706850994-6.jpg) 

5. 设置密码：至少8个字符，包含一个数字和一个大写字母。按要求完成设置后点击“下一步”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/baaa5fdcba_1706851219-7.jpg) 

6. 账户创建完成后点击“完成”。暂时跳过“通行秘钥”，它需要面部扫描并存储到Apple ID；熟悉后可在设置中添加。先点击“×”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8906412ffa_1706851584-8.jpg) 

# **二、进行身份认证**

币安要求必须完成身份认证，包括人脸识别步骤；否则无法使用服务。

1. 填写居住信息：如果您身处英国，选择英国会提示服务不可用。因此，即使您在英国，也应填写身份证上的原国籍地址，例如中国。点击“继续”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8e8c13ea48_1706852011-9.jpg) 

2. 填写姓名后点击“继续”。再填写国籍（如中国），点击“继续”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/aed64a7d7a_1706852271-10.jpg) 

3. 填写出生年月后点击“继续”。选择证件类型（身份证或护照均可），再次点击“继续”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/790a948131_1706852496-11.jpg) 

4. 拍摄证件正反面照片（因演示账户已认证，无法展示）。后续重点如下：

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/4276c0c5ea_1706852882-12.jpg) 

拍摄证件后，进行人脸识别。填写居住地址时，避免填写英国地址；使用证件上的信息即可。币安不会邮寄信件或要求地址证明。

# **三、安全设置**

1. 点击左上角的四个点，再点击右上角的“Pro”，将简洁版切换为Pro版。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/637cf1f0b1_1706853143-15.jpg) 

2. 再次点击左上角的四个点，认证完成后显示“已认证”。点击上方齿轮图标（设置）。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/ef0abc3041_1706853403-18.jpg) 

3. 进入“账户安全”查看设置。

手机号未绑定（无需绑定英国手机号）。

邮箱已在注册时绑定。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/5e29d66444_1706853975-19.jpg) 

4. 身份验证器（谷歌验证器）生成动态6位数字码，用于提币或交易时验证，提供额外安全保障。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/ec86c74fe7_1706854097-21.jpg) 

5. 通行秘钥将币安账户、手机和Apple ID绑定。提币或交易时需验证；如果手机丢失，他人无法通过面部扫描盗取资产，增强账户安全。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/8a70c78cd9_1706854434-23.jpg) 

# **四、购买数字货币**

1. 如需购买数字货币，点击首页“充值”，选择“快捷买币”或“数字货币充值”。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/c37c7beb23_1706854701-26.jpg) 

2. “数字货币充值”允许直接将USDT充值到币安交易所。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/c02c6e4ced_1706854873-27.jpg) 

3. 点击“快捷买币”时，只能使用身份证件对应的法定货币购买；不支持英镑交易。

![](https://pub-cf966733b80f4cb19e87f718fa1335dd.r2.dev/images/3c12e73c03_1706855260-29.jpg)
